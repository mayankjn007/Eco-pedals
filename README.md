# Eco - Pedals

This is a welcome page for Eco-Pedals Bike Sharing System.
Added Authentication for Management Dashboard.

### Version: 1.0.0

### Usage

```sh
$ npm install
```

```sh
$ npm start
# Or run with Nodemon
$ npm run dev

# Visit http://localhost:5000
```

### MongoDB

Open "config/keys.js" and add your MongoDB URI, local or Atlas
